# OpenCHAMI Quadlet Based Deployment Tool Compatibility Guide

As OpenCHAMI evolves, the procedures for deploying it evolve too. This
guide lists the versions of the deployment tool required for given
versions of OpenCHAMI/release. If the release of OpenCHAMI you want to
deploy is not listed here, you should use the latest version of the
tool, but be aware that there may be changes required to successfully
deploy OpenCHAMI.

- OpenCHAMI Release <= v0.1.4 should use Deployment Tool v0.0.1
